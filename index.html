🔍 Étapes de diagnostic

  • • 📱 Ouvrez la console: Sur mobile, allez dans Paramètres Chrome → Plus d'outils → Console développeur
  • • 🎤 Testez d'abord en anglais: Dites "Hello world" pour voir si ça fonctionne
  • • 🔄 Si échec en arabe: Le système basculera automatiquement en anglais
  • • 🌍 Problème fréquent: L'arabe n'est pas disponible sur tous les appareils
  • • 📞 Test rapide: Dites quelque chose et regardez les logs dans la console
  • • 🔧 Solutions: Chrome mobile → Menu → Plus d'outils → Console développeur
